## تحسينات الشعار والهوية
- إضافة شعار مؤسسة إبداع الظل في الهيدر والفوتر.
- إنشاء favicon.ico و apple-touch-icon و logo-512.png و logo-120.png.
- إضافة الشعار داخل البيانات المنظمة Schema عبر خاصية logo.
- إنشاء صورة مشاركة اجتماعية og-cover.png بمقاس 1200x630.

## تحسينات SEO التقنية
- عناوين Meta Title محسنة لكل صفحة.
- أوصاف Meta Description فريدة لكل صفحة.
- Canonical URL لكل صفحة.
- Hreflang ar-SA و x-default.
- Open Graph و Twitter Cards.
- Robots meta موسع: max-image-preview:large.
- Structured Data متقدمة:
  - LocalBusiness / HomeAndConstructionBusiness
  - WebSite
  - WebPage
  - Service
  - BreadcrumbList
  - FAQPage
  - ItemList في الصفحة الرئيسية
- Sitemap XML يحتوي lastmod وصور الصفحات الأساسية.
- robots.txt محدث مع رابط sitemap.
- 404.html مهيأة بتصميم وهوية الموقع.

## تحسينات الأداء
- إضافة width و height للصور لتقليل اهتزاز التخطيط CLS.
- إضافة loading lazy و decoding async للصور غير الأساسية.
- تحميل صورة hero بأولوية عالية في الصفحة الرئيسية.
- إضافة CSS خاص بتحسينات SEO والهوية.
- ضغط صور WebP الكبيرة عند الإمكان.

## ملاحظات مهمة بعد الرفع
1. ارفع الملفات إلى نفس الدومين: https://www.ebdadhel.com
2. افتح Google Search Console وأرسل:
