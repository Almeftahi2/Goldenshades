# موقع مؤسسة إبداع الظل (نسخة أولية جاهزة)

هذه نسخة موقع ثابت (Static) جاهزة للنشر.

## الملفات المهمة
- `index.html` الصفحة الرئيسية
- `mazallat.html` المظلات المودرن
- `sandwich.html` الساندوتش بانل
- `cement.html` الإسمنت بورد
- `majalis.html` المجالس والجلسات
- `contact.html` اتصل بنا
- `assets/css/style.css` التنسيق
- `assets/js/main.js` وظائف عامة + نموذج واتساب
- `assets/js/lightbox.js` تكبير الصور (Lightbox)
- `assets/img/` الصور
